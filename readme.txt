Repository voor Webdesign Module Masterpiece

Installeren bestanden:
Maak een map aan waarin alle bestanden komen te staan
Open deze map in Virtual Code Studio
Open een terminal. Controleer of het pad goed staat
Geef het volgende commando in: npm create vite@latest virtuele-kunstgalerij -- --template react
Kies bij de optie "Use rolldown-vite (Experimental)?" voor no (groen geselecteerd, bevestigen met "enter")
Kies bij de optie "Install with npm and start now?" voor yes (groen geselecteerd, bevestigen met "enter")
Er wordt nu een nieuwe map gemaakt met alle benodigde bestanden

De server zal ook automatisch worden gestart
Wanneer dit is gebeurd, kan deze worden geopend via de link: http://localhost:5173/

Zelf opstarten: 
Geef in de terminal waar de bestanden instaan het commando: npm run dev
In de terminal zal worden weergegeven dat de server wordt opgestart
Wanneer dit is gebeurd, kan deze worden geopend via de link: http://localhost:5173/
